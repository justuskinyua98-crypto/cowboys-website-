Team Admin SOP (Simple Operations)

Use this page as the daily checklist for updates, donations, and publishing workflow.

Safety Rule: Update through admin forms only. Do not edit code files unless approved by leadership.

Daily Content Workflow (10-20 mins)

  1. Open website and enter Admin Key.
  2. Upload photos/videos in Media & Team → Upload File To Server.
  3. Add new items in each manager:
    Outfits, Events, Livestock, Decor, Team, Social Links.
  4. Click matching Save ... To Backend button after each section.
  5. Refresh page and confirm cards, prices, and images are correct.

Donation SOP

  1. Donor submits donation intent in Foundation section.
  2. Admin opens Recent Donations (Admin).
  3. Copy donation ID and verify payment received (Till/Bank).
  4. Click confirm and enter payment reference code.
  5. Ask donor to download receipt from website.
  6. Weekly: download donations CSV and keep finance record backup.
Never share donor private data publicly. Keep donation records secure.

Fast Publishing Checklist

  • Photos clear and properly cropped
  • Event dates correct
  • Ticket and product prices verified
  • WhatsApp/contact links working
  • Search bar can find new items

Weekly Growth Tasks

  1. Export subscriber CSV.
  2. Use Campaign Builder for outfit drop and event update email.
  3. Post same campaign on social channels.
  4. Review most requested items and restock priority.

Escalation Rules

  • If payment confirmation fails: log ID + reference and notify admin lead.
  • If wrong image or price is published: correct immediately and resave to backend.
  • If site becomes slow/unreachable: restart server and notify technical lead.
