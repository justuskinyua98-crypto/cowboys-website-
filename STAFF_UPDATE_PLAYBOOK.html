Staff Update Playbook

Use this guide to update posters and stock without editing code files.

Rule: Never edit index.html, script.js, styles.css, or server.js. Use only the website admin forms.

Before You Start

  1. Open the main site and scroll to admin sections.
  2. Enter Admin Key in the admin key field.
  3. Prepare images with clean names (example: goat-premium.jpg).

How To Update Outfits

  1. Go to Admin Photo Manager.
  2. Upload image from Upload File To Server (folder: outfits).
  3. Copy uploaded path from Uploaded Path.
  4. Paste into Image Path or URL.
  5. Fill name, price, collection, description.
  6. Click Add Outfit.
  7. Click Save Outfits To Backend.

How To Update Event Posters

  1. Upload event poster (folder: events).
  2. Go to Admin Ticket Manager.
  3. Paste uploaded path into Poster/Flyer Image Path.
  4. Add title, date, venue, performers, ticket price.
  5. Click Add Event.
  6. Click Save Events To Backend.

How To Update Livestock

  1. Upload livestock image (folder: livestock).
  2. Go to Admin Livestock Manager.
  3. Paste path into Photo Path.
  4. Fill animal name, weight, rate/kg, status.
  5. Click Add Listing.
  6. Click Save Livestock To Backend.

Quality Checklist (Always)

  • Image appears correctly on card
  • Price is correct (KES)
  • No typing errors in names and dates
  • Saved to backend button clicked
  • Hard refresh page and re-check
If you can see the update after refresh, it is saved correctly.
